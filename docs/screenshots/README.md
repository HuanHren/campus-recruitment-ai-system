| 文件名 | 内容 |

| `04-company-dashboard.png` | 企业首页，展示岗位发布和简历筛选 |
| `05-job-list.png` | 岗位列表页，展示搜索筛选和岗位信息 |

| `07-interview.png` | 面试邀请页面 |
